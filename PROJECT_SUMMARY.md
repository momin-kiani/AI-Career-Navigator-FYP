# Career X - Final Year Project Summary

## Project Overview
**Career X** is an AI-powered career navigation platform designed to help users optimize their career paths through intelligent assessments, resume optimization, job matching, mentorship, and comprehensive market analytics.

## Technology Stack

### Backend
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- Bcrypt for password hashing
- Nodemailer for emails
- Multer for file uploads
- pdf-parse & mammoth for document parsing

### Frontend
- React.js
- React Three Fiber (3D)
- Framer Motion (Animations)
- Recharts (Charts)
- Tailwind CSS (Styling)
- Axios (API calls)

## How to Run

1. **Backend:**
   ```bash
   cd backend
   npm install
   node server.js
   ```

2. **Frontend:**
   ```bash
   cd frontend/frontend
   npm install
   npm install @react-three/fiber @react-three/drei three framer-motion recharts
   npm start
   ```

3. **Access:**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000
